About Company:

Collective Minds Gaming develops video game accessories and services. A company with award-winning has risen to top sellers at leading American retailers such as Amazon, Walmart, and Game Stop.

Description:

A game pack app designed to optimize games and get advantage of games, automate redundant part of games, and generate game coins. Using various languages, backend and frontend technologies, middleware, and server-client architecture.

Actions:

Development of Core Code: Created the core GamePack code in GPC (C-based) and Python, ensuring communication between embedded systems, mobile apps, and game controllers (Xbox/PS4).
Modularization: Broke down the GPC code into manageable modules for efficient development and easier maintenance.
Compilation and Debugging: Used a custom Python-based pre-compiler to merge modules, generate binary and JSON outputs, check for errors, and ensure optimized performance across platforms.
Front-End Development: Built custom web components (HTML, CSS, JavaScript) to create an intuitive UI for the Strikepack Central Mobile App.
Game-Specific Code: Implemented mod, generator, and macro codes for various games, using a scripting language tailored for game controllers.

Responsibilities:

• Code Integration & Development: Integrated and developed cross-platform GamePack code.
Implemented generator and mod codes in Gaming Programing Codes (GPC), a scripting language used by Cronus Zen game controller protocol, for several games on Zen using Studio, and our pre-compiler in Python
Implementing codes on different platforms, developing the core codes to our Game library, and connecting embedded systems Zen, Strikepack (Bluetooth devices), DS stick prototype, PS4, Xbox, and mobile app together in GPC (C-based language), Python, JavaScript, HTML, CSS

• Modularization & Error Checking: Broke down and debugged GPC code for error-free execution.

• Front-End & UI Development: Created and maintained web components for seamless user interaction.
Developing Web Components without any libraries for creating custom HTML tags in our front-end UI, designing the UI styles in CSS and templates in JavaScript to create our Android Game App using VSCode and Android Studio, and developing our own core library

• Optimization: Ensured that the GamePack optimized game performance and allowed for automation of repetitive tasks.

Result:

• Released Gamepack app in google store and app store that is worked with CM strike pack horizon
• Seamless Communication between controllers, mobile apps, and embedded systems.
• Cross-Platform Functionality was enhanced, allowing smoother integration between systems.
• Efficient Game Automation: The GamePack automated tasks in games like FIFA and Fortnite, saving time and improving the player experience.
