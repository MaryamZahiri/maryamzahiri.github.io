About Company:

Collective Minds Gaming develops video game accessories and services. A company with award-winning has risen to top sellers at leading American retailers such as Amazon, Walmart, and Game Stop.

Description:

Specialized in enhancing gaming experiences through innovative products and applications. The game products optimize gameplay with customizable options such as paddles and back buttons, and protocols that transform bare systems into elite ones along with advanced gamepack features like mod and macro capabilities via apps.

Actions:

Integration of Firmware and Bluetooth Stack: Integrated RTOS for STM32 (ARM-based) firmware, with Bluetooth stack support using UART, USB interfaces, drivers, and MCU configurations.
Bootloader Development: Developed a bootloader in C for SAMD21 Atmel to handle communication via USB and UART interfaces.
Cross-Platform Code Conversion: Converted bare metal back button code from STM32 to the SAM platform using I2C interface for enhanced functionality.
Application Development: Created both client and server-side Bluetooth chat applications, facilitating communication between devices via mobile apps and embedded systems.
Hardware-Software Integration: Built a PS4 strike pack using BlueNRG and STM32, integrated with I2C interface for master-slave communication, and debugged it via ST-link.

Responsibilities:

Firmware Development: Developed and integrated RTOS, bootloaders, and Bluetooth communication protocols for embedded platforms.
Cross-Platform Code Adaptation: Converted and optimized codes for different ARM-based platforms.
Software Development: Created mobile and embedded applications for communication between hardware systems.

How:

Goal:

The goal was to enhance the gaming experience by providing users with customizable options that offer improved control and game automation features. By integrating various hardware and software systems, the aim was to optimize gameplay and make it easier for players to focus on more engaging aspects while enhancing their competitive advantage.

Result:

• Enhanced Game Control Features: The game products included advanced features like customizable paddles and back buttons for a superior gaming experience.
• Optimized Gameplay: Provided mod and macro capabilities through applications, allowing users to automate repetitive tasks and improve game performance.
• Cross-Platform Compatibility: Ensured seamless communication between various embedded systems and mobile applications.
