Description What I did:

Development of Human-tracking Robot using the QR Code recognition in Linux environment on Intel NUC. ROS-based processes for mobile robots (powerbot)
One main issue in mobile robotics is the task of localization and navigation. A human-tracking capability for a Powerbot using the landmark as a position reference is proposed as one approach to this problem. The Powerbot can follow a person, an object or another robot as a leader. In this method, a QR code on the leader is determined by the Powerbot. The Powerbot with a mounted camera tracks the position and orientation of the leader’s QR code at high speed. There are three steps in the human-following process. First, a leader marked with a QR code is recognized. Second, location detection of the QR code applying a shape-based pattern matching algorithm recognizes the position of the QR code. Third, the Powerbot maintains a determined distance from the leader while following. The third step is a robot-control experiment to demonstrate the accuracy of the tracking algorithm.

Actions:

QR Code Detection: A mobile robot with a mounted camera uses a vision-based algorithm to detect and recognize QR codes. The QR code on a leader (a person or another robot) serves as a landmark for localization and navigation.
Location Detection: A shape-based pattern matching algorithm is applied to accurately determine the location of the QR code and, thus, the position of the leader.
Robot Control: The robot adjusts its speed and maintains a fixed distance from the leader, ensuring smooth tracking by following the QR code in real-time.

Responsibilities:

Algorithm Development: Designed and implemented a high-accuracy vision system for QR code detection, focusing on fast recognition and robust navigation.
System Integration: Integrated the vision-based QR code tracking system with the robot control mechanism to ensure accurate localization.
Testing and Debugging: Conducted experiments to evaluate the performance of the human-following robot and fine-tuned the system for real-time functionality.

How:



The QR code detection for a human-tracking robot was made using a combination of software technologies and a camera-based vision system using vision-visp. Here's a breakdown of how it worked:

Software and Technology Involved:
• Camera and Vision System:
The robot was equipped with a camera to capture real-time images. The QR code was used as a landmark for localization and navigation.
• Image Processing Algorithms:
A shape-based pattern matching algorithm was used to identify and detect the QR code in the captured images.
- The algorithm involved:
Converting the image to grayscale for better accuracy.
Edge detection and grid-cell generation to analyze the QR code's structure.
The QR code's position and orientation were determined based on its recognition in the image, allowing the robot to understand the leader’s location.
• Robot Operating System (ROS):
The robot control part was integrated into a ROS-based framework, ensuring that processes like localization, navigation, and movement control could be managed effectively.
• Control Algorithms: A control algorithm was implemented to maintain a specific distance from the leader (the human or object being tracked). It ensured that the robot followed the leader’s movements without deviating from the path.

How the Detection Worked:
Step 1: The camera on the robot captures the image of the environment.
Step 2: The image is processed using pattern-matching techniques to detect the QR code in the frame.
Step 3: Once the QR code is identified, the algorithm calculates its position and orientation in the robot’s reference frame.
Step 4: The robot uses this information to adjust its movements, ensuring it follows the QR code (leader) while maintaining a set distance.

Goal:

QR code detection aimed to solve the problem of localization and navigation in mobile robotics, particularly in dynamic environments. By using QR code recognition, the robot can follow a leader in real-time, making the system suitable for applications requiring autonomous following, such as warehouse robots or service robots. The choice of QR codes allows for cost-effective and scalable solutions for localization.

Results:

Accurate Tracking: The robot successfully followed the leader using QR code detection, with high-speed recognition and precise distance maintenance.
Improved Localization: The use of QR codes enhanced the accuracy of localization and navigation, solving common issues related to non-homogeneous lighting and real-time movement.
Obstacle Avoidance: By integrating the system with ultrasonic sensors, the robot avoided obstacles, maintaining smooth tracking even in dynamic environments.
